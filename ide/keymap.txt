快捷键 用途描述

Find KeyMap

Ctrl + Shift + F 		find a string　
Ctrl + Alt + H	 		find usage
Ctrl + n 类名查找  		Ctrl + shift + n文件查找　　　double-shift 	Search EveryWhere 
Ctrl + L 查找下一处 	CTRL + SHift + L	前一处 

Move KeyMap

Ctrl + E 最近修改的文件　	Alt + left/right 切换代码视图 
Ctrl + Alt + left/right 	返回至上次浏览的位置
Ctrl + Shift + Backspace 	可以跳转到上次编辑的地方 
Alt + Up/Down 		在方法间快速移动定位 
Ctrl + [/] 	  		可以跳到大括号的开头结尾
Crtl + G 	  		jump to Line 　
F2/Shift + F2  		高亮错误或警告快速定位/Jump to Error　
Alt + Shift + C 	最近更改的代码
Ctrl + Shift + Up/Down 	代码向上/下移动
Ctrl + B 		Jump To Declaration
Ctrl + Alt + B 	Jump To Implementation

Fix KeyMap

Alt + 回车 导入包,自动修正 　 
Ctrl + Alt + L 格式化代码　　　　　　Ctrl + Alt + O 优化导入的类和包　　
Alt + Insert 			生成代码(如get,set方法,构造函数等)　
Ctrl + D 复制行 　　　  Ctrl + X 剪切行 Ctrl + Y 刪除hang 
Ctrl + Alt + T 			生成if-else/try-catch　
Ctrl + Shift + Space 	自动补全代码 Ctrl + space 代码提示 
Ctrl + P 方法参数提示 	Ctrl + Q 显示注释文档 　
Shift + F6 重构-重命名
Ctrl + / 行注释 		Ctrl + Shift + / 代码段注释　

　　　　　　　 
ctrl + alt + u 	类层次UML图　 　　　　　　　　　　　　　　　
Ctrl+U 			导航到父类 Ctrl+O 重写或实现父类　　　　　　　　　　　　　　　　　　　　　　　　　　　　　　
Ctrl + T 		Update project from VCS
ctrl + w 		每次按下都会扩大选中的区域 
Alt + 1 		显示目录窗口	
alt + F12 		打开终端	 ctrl+shift+F12 关闭工具栏
